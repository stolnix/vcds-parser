# Privacy Policy for VCDS Parser

**Effective date:** June 26, 2026

VCDS Parser is an app used to parse, view, and search through VCDS `.txt` diagnostic log files.

This Privacy Policy explains how VCDS Parser handles information.

## 1. Data Collection

VCDS Parser does **not** collect, transmit, store, sell, or share any personal data.

The app does not collect:

* names
* email addresses
* phone numbers
* location data
* device identifiers
* analytics data
* crash analytics
* advertising identifiers
* usage tracking data
* uploaded files
* diagnostic log contents

## 2. Local File Processing

VCDS Parser processes VCDS diagnostic text files locally on the user’s device.

When a user opens or imports a VCDS `.txt` log file, the file is used only for local parsing, viewing, and searching inside the app. The contents of the file are not uploaded to any server and are not sent to the developer or to any third party.

## 3. No User Accounts

VCDS Parser does not require users to create an account, log in, or provide personal information.

## 4. No Internet-Based Data Transfer

VCDS Parser does not transmit user data to external servers.

The app does not use user data for analytics, advertising, profiling, tracking, or marketing.

## 5. Third-Party Services

VCDS Parser does not use third-party analytics, advertising networks, tracking SDKs, or cloud services that collect user data.

## 6. Data Sharing

Because VCDS Parser does not collect user data, it does not share user data with third parties.

## 7. Data Retention

VCDS Parser does not retain user data on external servers.

Any VCDS files opened by the user remain under the user’s control on their device or in the location from which the user selected the file.

## 8. Children’s Privacy

VCDS Parser does not knowingly collect personal information from children or from any other users.

## 9. Changes to This Privacy Policy

This Privacy Policy may be updated if the app’s functionality or data practices change. Any changes will be posted by updating this Privacy Policy.
