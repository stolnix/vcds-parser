# VCDS Parser Support

VCDS Parser is an iOS app for opening, parsing, viewing, and searching through VCDS `.txt` diagnostic log files.

## What the App Does

VCDS Parser helps users inspect VCDS diagnostic text logs more easily by allowing them to parse and search through the contents of supported `.txt` files.

## Supported Files

VCDS Parser is intended for plain text VCDS diagnostic log files using the `.txt` file format.

If a file does not open or parse correctly, check that:

* the file is a valid VCDS text log
* the file has not been corrupted
* the file is not password protected or encrypted
* the file contents are plain text

## Data and Privacy

VCDS Parser processes files locally on the user’s device.

The app does not collect, upload, transmit, sell, or share user data.

## Troubleshooting

If the app does not display the expected log contents:

1. Confirm that the selected file is a VCDS `.txt` diagnostic log.
2. Try opening the file in a text editor to confirm that the content is readable.
3. Make sure you are using the latest available version of VCDS Parser.
4. Restart the app and try importing the file again.

## Contact

For support, questions, or bug reports, open the issue here.

When reporting an issue, please include:

* app version
* iOS version
* device model
* a short description of the problem
* whether the issue happens with one file or all files

Do not send diagnostic logs if they contain private or sensitive information unless you are comfortable sharing them.
